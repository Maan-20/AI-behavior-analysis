## ⭐ Featured Evaluation

### 🧠 Reasoning Consistency Under Prompt Variation  
👉 [View Full Analysis](reasoning_consistency_test.md)

This evaluation examines how slight changes in prompt wording affect the model’s ability to apply logical reasoning consistently. Despite identical underlying logic, the model produces different answers, revealing reliance on pattern recognition rather than true reasoning.

**Key Insight:**  
The model demonstrates instability in multi-step reasoning tasks, indicating sensitivity to prompt phrasing and a tendency to rely on heuristics instead of structured logic.
